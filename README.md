# FLYTAU  | Flight Management Web Application

## About The Project
FLYTAU is a comprehensive flight management and booking web application, developed as a 2nd-year academic project for the Industrial Engineering and Management B.Sc. program at Tel Aviv University. 

The system simulates a real-world flight booking flow, including database management, user interaction, seat selection, and checkout processes.

### Built With
* **Backend:** Python, Flask
* **Database:** MySQL
* **Frontend:** HTML, CSS

## Repository Structure
This repository contains all the necessary files to run the application and set up the database:

* `pythonProject/`: Contains the core application source code, including Python main scripts, routing, and the `templates` (HTML) and `static` (CSS) folders.
* `FLYTAU_DB.sql`: The SQL script required to install, initialize, and populate the system's database.
* `ERD & TABLES/`: Contains the database design planning and Entity-Relationship Diagrams.
* `דוחות ניהוליים/` & `Group_46.docx`: Academic project documentation and management reports (in Hebrew).
## Getting Started
To run this project locally, you will need to set up a local MySQL server, run the `FLYTAU_DB.sql` script to create the schema, and execute the main Python script within the `pythonProject` directory using a Flask environment.
